Getting Started

Welcome to the Wormhole Queue System wiki! This guide will help you get up and running with our queue management platform.

Creating Your Account

To get started, create an account by hovering over the "User" button in the navigation bar. A dropdown will appear, click "Log In". Provide your email address and password.

Student Guide

Getting Help Online

On the "Home" page, there is a section called "For Students." First, submit a help request by clicking the link. Enter a name that the Wormhole Assistant will be able to identify on your platform of choice. Choose your class from the dropdown. Select the platform through which an assistant can reach you. You will automatically be redirected to the "Live Queue" page. Here you can see your place in line. Navigate back to the For Students section on the Home page to access the Zoom or Teams of the wormhole assistants.

In-person Help

Find a table with a button box on it. Simply press the button that corresponds to the class you need help with. A ticket with your table name and the class will be added to the queue displayed on the TV and a wormhole assistant will be with you when they are available.

Website Overview

Home

The Home page provides an overview of the Wormhole, including quick links to key features and resources. It shows the schedule for the Wormhole as well as the breakdown of wormhole assistant who will be available.

Viewing the Live Queue

The live queue page shows real-time information about the current queue status. Tickets are handled first come first serve, and in progress tickets will have an "in progress" label instead of a number in line.

Wiki

This wiki page provides comprehensive documentation, acting as a guide for using the Wormhole Queue System.

User

By hovering over the "User" button, you will open a dropdown menu of options. You do not need to login as a student to get physics help.

Wormhole Assistants

Wormhole Assistants can log in to manage their profile, view queue activity, and help students through the live queue workflow.

Admin Features

Admin users can manage users, review queue data, export archives, edit public website content, and monitor hardware status.

Troubleshooting

If a page does not load as expected, refresh the browser, confirm you are using the correct account, and contact an administrator if the issue continues.

FAQ

Do students need to log in?

No. Students can submit help requests without logging in.

Where can I see my place in line?

Use the Live Queue page to view the current request order.
